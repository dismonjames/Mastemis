# Deployment responsibilities

The Mastemis project maintains secure defaults, documented boundaries, validation, and remediation of reported defects. Operators secure hosts, TLS, PostgreSQL, object storage, backups, retention, administrator bootstrap secrets, compiler images, network segmentation, and local legal compliance. Invigilators review context and confirm violations rather than treating raw signals as guilt. Candidates must receive the required privacy notice before examination monitoring begins.

The local compose definition is for development. Its fallback credential and unencrypted local port are not production configuration. Production operators must configure TLS, a non-default PostgreSQL secret, durable source storage permissions, migration policy, administrator bootstrap through a secret provider, cookie lifetime, retention, backups, and monitoring of failed/poison outbox rows and expired worker leases. Bootstrap secrets should be removed immediately after successful initialization.
